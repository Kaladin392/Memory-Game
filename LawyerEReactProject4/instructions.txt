1. Install Node.js ---> https://nodejs.org/en/download/package-manager

2. In a command prompt, type "ntm install mongodb" and press enter.

3. In a command prompt, type "ntm install express" and press enter.

4. In a command prompt, type "ntm install mongoose" and press enter.

5. In a command prompt, navigate to the LawyerEReactProject4 folder and type "node server.js" and press enter.

6. Open the index.html page
